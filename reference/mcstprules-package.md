# mcstprules: Cost Allocation Rules in Minimum Cost Spanning Tree Problems

A comprehensive set of tools to compute, analyze, and visualize cost
allocation rules for Minimum Cost Spanning Tree Problems (MCSTP).

## Details

This package implements two major approaches for cost allocation found
in the literature: algorithmic rules and rules defined through
cooperative games. Additionally, it provides a robust suite of
analytical tools to evaluate core stability, geometric properties, and
cost sensitivity.

For a detailed overview, classification, and references of all
implemented rules and analysis tools, please see the central
documentation page:
[`mcstRules`](https://luciasouto13.github.io/mcstprules/reference/mcstRules.md).
